This prototype includes improvements to add questions to the form to enable SERCO to determine the order type description.

It also adds the new released under P-RARR question. This is a separate requirement but affects the same pages.

View the SERCO order type description mapping spreadsheet to see mapping that the service needs to solve.

The new fields that need to be added are:

  • Notifying organisation (needed to conditionally display name fields)
  • Notifying organisation name (only relevant for prisons, crown courts and courts Notifying Organisation types)
  • Responsible organisation region (only relevant for youth and probation Responsible Organisation types)
  • ISSP question
  • Sentence Type (only relevant for Post release Order type)
  • Form type (can be inferred from other questions)
  • Released under P-RARR question (only relevant for Post release Order type)

Key pages MVP:

  • Interested parties
  • Monitoring conditions

Key pages day 2:

  • Notifying organisation's details
  • Responsible Organisation's details
  • Responsible Organisation's region (conditional)
  • Order information
  • Conditional journey for order information
